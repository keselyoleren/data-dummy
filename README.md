Data dummy generator
=====


Random name, username, email, phone, address, ipv4 generator


Installation
------------

install with pip::

    pip install data-dummy==0.1


Usage
-----
Here are examples of all current features:

  
    >>> import dummy
    >>> dummy.name()
    'Yuliana Sudradjat'
    >>> dummy.username()
    'Yuliana'
    >>> dummy.email()
    'Bernard@gmail.com'
    >>> dummy.address()
    'Jl. Dr A Rivai Bukittinggi Telp: 0752-21720'
    >>> dummy.phone()
    '+62812-0989-0128'
    >>> dummy.ipv4()
    '123.124.5.12'
